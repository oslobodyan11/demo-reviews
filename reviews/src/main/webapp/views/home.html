Demo Library Reviews App - Single Page Application with Spring RESTful backend

Here is the home page of DEMO Library Reviews application. There are few menu links on the side navigation. Click on the links is loading the corresponding content to the main rendering area. All interactions between client and server are implemented with AngularJS and RESTFull web services. For the DEMO no data is stored in the database.

Implementation Notes

  • Home Page Link loads this page
  • Library Link retrieves the book list from the server.
  • New Book Link submits new book, the data is sent to server using ngResource service, $resource (save action method)
  • DEMO uses ng-view directive to render different views
  • ngRoute module is used to load different views. Controller and URL are configured for each view
